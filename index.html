01. JavaScript හැඳින්වීම සහ මූලිකාංග

1.1 Scripting Language එකක් යනු කුමක්ද?

ස්ක්‍රිප්ටින් ලැන්ග්වේජ් එකක් යනු වෙනත් මෘදුකාංගයක් (උදා: Web Browser එකක්) පාලනය කිරීමට හෝ යම් කාර්යයන් ස්වයංක්‍රීය කිරීමට (Automation) භාවිතා කරන ප්‍රෝග්‍රැමින් ලැන්ග්වේජ් වර්ගයකි.

ප්‍රධාන ලක්ෂණ:

  • Interpreted: මේවා Compile කරන්නේ නැත. Interpreter එකක් මගින් කේතය පේළියෙන් පේළිය කියවා එසැණින් ක්‍රියාත්මක කරයි.
  • Dynamic Behavior: වෙබ් පිටු වලට සජීවී බවක්, චලනයන් සහ පරිශීලක ප්‍රතිචාර (Interactivity) ලබා දීමට භාවිතා කරයි.
  • Examples: JavaScript, Python, PHP, Ruby.

1.2 Client-Side Scripting

Client-side scripting යනු පරිශීලකයාගේ වෙබ් බ්‍රව්සරය (Browser) තුළ ක්‍රියාත්මක වන කේත වේ. මෙය සර්වර් එකේ (Server) ක්‍රියාත්මක නොවේ.

මෙයින් කළ හැකි දේ:

  • DOM එක වෙනස් කිරීම (පිටුවේ ඇති අකුරු, පාට, layout මාරු කිරීම).
  • Form Validate කිරීම (සර්වර් එකට යැවීමට පෙර දත්ත නිවැරදිදැයි බැලීම).
  • User Events හැසිරවීම (Click කිරීම්, Typing කිරීම් වලට ප්‍රතිචාර දැක්වීම).

1.3 JavaScript හි ඉතිහාසය

1995 දී Brendan Eich විසින් Netscape සමාගමේදී මෙය නිර්මාණය කරන ලදී. මෙහිදී මතක තබා ගත යුතු වැදගත්ම කරුණ වන්නේ JavaScript සහ Java යනු සම්පූර්ණයෙන්ම වෙනස් භාෂා දෙකක් බවයි. හුදෙක් මාර්කටිං අවශ්‍යතාවය මත පමණක් JavaScript යන නම භාවිතා විය.

1.4 JavaScript හි ප්‍රධාන විශේෂාංග

  • 1. Dynamic Typing: Variable එකක් හදන විට එහි වර්ගය (Number/String) කලින් ප්‍රකාශ කළ යුතු නැත. එහි අගය අනුව වර්ගය තීරණය වේ.
  • 2. Case Sensitive: අකුරු වල හැඩය (Capital/Small) අනුව variables එකිනෙකට වෙනස් ලෙස සලකයි. උදා: myVal සහ myval දෙකකි.
  • 3. Interpreted Language: කේතය පේළියෙන් පේළිය බ්‍රව්සරය මගින් කියවා ක්‍රියාත්මක කරයි.

1.5 JavaScript ප්‍රතිදාන ක්‍රම (Output Methods)

JavaScript මගින් අපට තොරතුරු පෙන්විය හැකි ආකාර 5 ක් දේශන වල සඳහන් වේ:

// 1. Alert Box - Popup පණිවිඩයක් පෙන්වීමට window.alert("Welcome to our Lab!"); // 2. document.write() - කෙලින්ම HTML පිටුවට ලිවීමට document.write("This text is added to the page."); // 3. innerHTML - පවතින HTML element එකක් ඇතුළට අලුත් HTML ලිවීමට document.getElementById("myPara").innerHTML = "<b>Bold Text</b>"; // 4. console.log() - බ්‍රව්සරයේ console එකේ තොරතුරු පෙන්වීමට (Debugging වලට) console.log("Developer notice: The sum is " + (10+20)); // 5. textContent - HTML tags නැතිව plain text පෙන්වීමට (වඩා ආරක්ෂිතයි) document.getElementById("demo").textContent = "<b>This will show as text</b>";
පරෙස්සම් වන්න! පිටුව Load වුනාට පස්සේ document.write() භාවිතා කළොත් සම්පූර්ණ පිටුවම මැකී ගොස් අලුත් අගය පමණක් දිස්වේ.

1.6 විචල්‍යයන් (Variables): var, let, const

JavaScript හි දත්ත ගබඩා කිරීමට විචල්‍යයන් භාවිතා කරයි. මෑත කාලීනව `var` වෙනුවට `let` සහ `const` භාවිතය නිර්දේශ කරයි.

var x = 10; // පැරණි ක්‍රමය (Function scope) let y = 20; // නවීන ක්‍රමය (Block scope) const z = 30; // අගය වෙනස් කළ නොහැකි විචල්‍යයන් (Constants) // වැදගත්: const අගයන් පසුව වෙනස් කළ නොහැක. // z = 50; <-- Error එකක් එයි!

02. අරාවන් (Arrays) සහ ශ්‍රිත (Functions)

2.1 අරාවන් (Arrays)

අරාවක් යනු දත්ත සමූහයක් එකම නමකින් ගබඩා කළ හැකි පෙට්ටියක් වැනි ව්‍යුහයකි. මෙහි ඇති එක් එක් අගය "Element" එකක් ලෙස හඳුන්වයි.

Index (පටුන):

Array එකක අගයන් පටන් ගන්නේ සැමවිටම 0 සිටයි. එනම් පළමු අගය ලබා ගැනීමට 0 ද, දෙවන අගයට 1 ද භාවිතා කළ යුතුය.

// Array එකක් සාදන ක්‍රම let fruits = ["Apple", "Mango", "Banana"]; let numbers = new Array(1, 2, 3, 4, 5); // මිශ්‍ර දත්ත ද දැමිය හැකිය let mixed = [1, "Hello", true]; // අගයන් ලබා ගැනීම console.log(fruits[0]); // Apple console.log(fruits.length); // 3 (Array එකේ ඇති අගයන් ගණන)

2.2 අරාවන් හැසිරවීමේ ක්‍රම (Array Methods)

Array එකක දත්ත එකතු කිරීමට හෝ ඉවත් කිරීමට විශේෂ ක්‍රම කිහිපයක් ඇත:

let cars = ["BMW", "Audi"]; cars.push("Tesla"); // අගට එකතු කරයි: ["BMW", "Audi", "Tesla"] cars.unshift("Ford"); // මුලට එකතු කරයි: ["Ford", "BMW", "Audi", "Tesla"] let last = cars.pop(); // අග එක ඉවත් කර ලබා දෙයි (Tesla ඉවත් වේ) let first = cars.shift(); // මුල් එක ඉවත් කර ලබා දෙයි (Ford ඉවත් වේ) // slice(start, end) - කොටසක් වෙන් කර ගනී (මුල් එක වෙනස් නොවේ) let sub = cars.slice(0, 1);

2.3 Array Iteration (පරිලෝකනය)

Array එකක හැම අගයක්ම එකින් එක පරික්ෂා කරන ආකාරය:

let marks = [50, 75, 90]; // 1. සාමාන්‍ය For loop එකකින් for(let i=0; i < marks.length; i++) { console.log(marks[i]); } // 2. for...of loop (වඩාත් පහසු ක්‍රමය) for(let m of marks) { console.log(m); } // 3. forEach method එකෙන් marks.forEach((m, idx) => { console.log("Index " + idx + " score is " + m); });

2.4 map() සහ filter() ක්‍රම

map(): Array එකක ඇති හැම අගයක්ම වෙනස් කර (Transform) අලුත් array එකක් ලබා දෙයි.

const prices = [100, 200, 300]; const discounted = prices.map(p => p * 0.9); // 10% discount: [90, 180, 270]

filter(): යම් කොන්දේසියකට (Condition) ගැලපෙන අගයන් පමණක් වෙන් කර අලුත් array එකක් ලබා දෙයි.

const ages = [15, 22, 10, 30]; const adults = ages.filter(a => a >= 18); // [22, 30]

2.5 ශ්‍රිත (Functions) හැඳින්වීම

ශ්‍රිතයක් යනු යම් කාර්යයක් කිරීමට ලියන ලද කේත කොටසකි. මෙය අපට අවශ්‍ය ඕනෑම විටක භාවිතා (Call) කළ හැකිය.

// 1. Function Declaration (සාමාන්‍ය ක්‍රමය) function sayHello(name) { console.log("Hello, " + name); } // 2. Function Expression (Anonymous - නමක් නැති ශ්‍රිත) const greet = function(msg) { console.log(msg); }; // Function call කිරීම sayHello("Kasun");

03. Arrow Functions, Async JS සහ Objects

3.1 Arrow Functions (ES6 Feature)

කෙටි Syntax එකකින් ශ්‍රිතයක් ලිවීමට මෙය භාවිතා කරයි. මීට "Fat Arrow" කියාද කියයි.

// පැරණි ක්‍රමය const add = function(a, b) { return a + b; }; // Arrow function ක්‍රමය const addArrow = (a, b) => a + b; // පරාමිතීන් (Parameters) එකක් නම් වරහන් අවශ්‍ය නැත const square = x => x * x;

3.2 Asynchronous JavaScript

JavaScript යනු Single-threaded භාෂාවකි. එනම් වරකට එක වැඩක් පමණක් කරයි. නමුත් සමහර වැඩ (උදා: File download කිරීම) අවසන් වන තුරු අනිත් වැඩ නතර නොවී සිදු වීමට මෙය ඉඩ සලසයි.

කෝපි හලක උපමාව (Cafe Analogy):

ඔබ කෝපි එකක් ඇණවුම් කර (Async task start), එය ලැබෙන තුරු නොසිට වෙනත් වැඩක යෙදී (Other JS code), කෝපි එක සූදානම් වූ පසු එය ලබා ගැනීම (Callback) Asynchronous ක්‍රියාවලියකි.

// setTimeout: කාලයකට පසු වැඩක් කිරීමට console.log("Start"); setTimeout(() => { console.log("Wait for 3 seconds..."); }, 3000); console.log("End"); // Output එක වෙන්නේ: Start, End, ඉන්පසු Wait for 3 seconds...

3.3 Callback Functions

ශ්‍රිතයක් තවත් ශ්‍රිතයකට argument එකක් ලෙස යැවීම Callback එකක් ලෙස හඳුන්වයි. යම් කාර්යයක් අවසන් වූ පසු Callback එක ක්‍රියාත්මක වේ.

3.4 වස්තූන් (Objects)

වස්තුවක් (Object) යනු දත්ත සහ ඒවායේ අගයන් එකතුවකි (Key-Value pairs). සැබෑ ලෝකයේ ඇති ඕනෑම දෙයක් මෙලෙස ගබඩා කළ හැකිය.

const person = { firstName: "Nimal", age: 25, city: "Matara", // Method එකක් (Object එකක් ඇතුළත ඇති ශ්‍රිතයක්) greet: function() { console.log("Hi, I am " + this.firstName); } }; // දත්ත ලබා ගැනීම console.log(person.firstName); // Dot notation console.log(person["age"]); // Bracket notation

3.5 "this" Keyword එක

`this` යනු වස්තුවක් තුළ සිට එම වස්තුවටම ආමන්ත්‍රණය කිරීමට භාවිතා කරන විශේෂ වචනයකි.

3.6 Factory Functions

එකම ආකාරයේ objects ගොඩක් නිර්මාණය කිරීමට භාවිතා කරන ශ්‍රිත මේ නමින් හැඳින්වේ.

function createStudent(name, rNumber) { return { name: name, rollNo: rNumber, display() { return `Name: ${this.name}, Roll: ${this.rollNo}`; } }; } const s1 = createStudent("Amal", 101); const s2 = createStudent("Kamal", 102);

04. DOM මෙහෙයවීම සහ සිද්ධි (Events)

4.1 DOM (Document Object Model)

DOM යනු HTML පිටුවක ඇති හැම element එකක්ම JavaScript වලට තේරෙන "Object" එකක් විදිහට පෙන්වන ගසක් (Tree structure) වැනි ව්‍යුහයකි. JS හරහා අපට මෙම ගසේ ඇති දේවල් මාරු කිරීම, මැකීම හෝ අලුතින් එක් කිරීම කළ හැකිය.

4.2 Elements ලබා ගැනීම (Selecting Elements)

HTML පිටුවේ ඇති දේවල් JS වලට අල්ලා ගැනීමට ක්‍රම කිහිපයක් ඇත:

// 1. ID එක මගින් (වඩාත් වේගවත්) const para = document.getElementById("paraID"); // 2. Query Selector (වඩාත් බලවත් - CSS Selector භාවිතා කළ හැක) const firstBtn = document.querySelector(".btn-primary"); const allItems = document.querySelectorAll("li"); // සියලුම li items ලබා ගනී

4.3 සිද්ධි (Events)

User කෙනෙක් පිටුව මත කරන ක්‍රියාකාරකම් Events ලෙස හඳුන්වයි. (උදා: Click කිරීම, Mouse එක උඩින් ගෙන යාම, අකුරු ටයිප් කිරීම).

Common Events:

  • click: බොත්තමක් එබූ විට.
  • mouseover: Mouse pointer එක element එකක් උඩට ආ විට.
  • change: Input එකක අගය වෙනස් වූ විට (උදා: Dropdown එකක්).
  • submit: Form එකක් submit කළ විට.

4.4 Event Listeners

යම් event එකක් සිදු වූ විට code එකක් run කරවීමට මෙය භාවිතා කරයි.

const myBtn = document.querySelector("#submit"); function handleAction() { alert("Form submitted successfully!"); } // Event listener එකක් එක් කිරීම myBtn.addEventListener("click", handleAction);

4.5 "defer" Attribute එකේ අවශ්‍යතාවය

විභාගයට වැදගත්!
HTML පිටුව සම්පූර්ණයෙන් load වීමට පෙර JS run වුවහොත්, `querySelector` මගින් element එක සොයාගත නොහැකි වී null error එකක් එයි. මෙය වැළැක්වීමට <script src="..." defer> ලෙස භාවිතා කළ යුතුය.

4.6 සම්පූර්ණ ප්‍රායෝගික අභ්‍යාසය (Exam Level Question)

Dropdown එකකින් වාහනයක් තෝරා බොත්තම එබූ විට paragraph එකක ඒ නම පෙන්වන code එක:

// script.js const btn = document.querySelector("#btn"); const select = document.querySelector("#carSelect"); const output = document.querySelector("#result"); btn.addEventListener("click", () => { let selectedCar = select.value; output.textContent = "You selected: " + selectedCar; output.style.color = "blue"; });

අමතර ප්‍රායෝගික අභ්‍යාස (Extra Practice)

1. Arrow Function අභ්‍යාසය

සංඛ්‍යාවක් (Number) ලැබුණු විට එය 0 ට වඩා වැඩි නම් "Positive" ලෙසත්, අඩු නම් "Negative" ලෙසත් return කරන Arrow Function එකක් ලියන්න.

2. Object අභ්‍යාසය

Employee කෙනෙකුගේ නම, පැයකට වැටුප (Hourly rate) සහ වැඩ කළ පැය ගණන ඇතුළත් object එකක් සාදන්න. ඔහුගේ සම්පූර්ණ වැටුප ගණනය කර දෙන method එකක් එයට ඇතුළත් කරන්න.

3. DOM අභ්‍යාසය

බොත්තමක් එබූ විට පිටුවේ පසුබිම් වර්ණය (Background Color) රතු පැහැයට වෙනස් වන code එක ලියන්න.
